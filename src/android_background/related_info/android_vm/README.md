# 安卓虚拟机

* 历史背景
  * Android
    * 代码语言：`Java`
      * `Java`的虚拟机是：`JVM`
  * Android：出于性能考虑，没用`JVM`，用了自己的虚拟机`VM`
* `安卓虚拟机`=`Android虚拟机`=`Android VM`
  * 旧：`Android < 5.0`：`Dalvik`
  * 新：`Android >= 5.0`：`ART`
